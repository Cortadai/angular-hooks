Este es un proyecto educativo de Angular 14 llamado **life-cycle-app** diseñado específicamente para enseñar y demostrar el funcionamiento de los hooks del ciclo de vida de componentes en Angular. La aplicación implementa todos los hooks principales (`ngOnInit`, `ngOnChanges`, `ngDoCheck`, `ngAfterContentInit`, `ngAfterContentChecked`, `ngAfterViewInit`, `ngAfterViewChecked`, y `ngOnDestroy`) con mensajes de consola que permiten visualizar el orden y momento de ejecución de cada uno. Incluye funcionalidades interactivas como un botón para mostrar/ocultar contenido, un campo de texto con enlace bidireccional (ngModel), un temporizador que cuenta segundos usando RxJS, y un componente hijo que demuestra el paso de propiedades y detección de cambios. El propósito principal es servir como herramienta de aprendizaje para desarrolladores que necesitan entender cómo y cuándo se ejecutan los diferentes hooks del ciclo de vida en Angular, así como las mejores prácticas para inicializar recursos y limpiarlos adecuadamente (como la cancelación de suscripciones en `ngOnDestroy`).
